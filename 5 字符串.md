# 36：不理解 rune 的概念

``` go
func main() {
	s := "hello"
	fmt.Println(len(s)) // 5

	s = "汉"
	fmt.Println(len(s)) // 3

	s = string([]byte{0xE6, 0xB1, 0x89})
	fmt.Printf("%s\n", s) // 汉
}
```

- Go 的源代码使用 UTF-8 编码，因此，所有字符串字面量都是 UTF-8 编码；
- rune 对应于 Unicode 代码位的概念，表示单个值表示的项，并且它可以由多个字节组成；
- 使用 UTF-8，一个 Unicode 代码位可以编码成 1 到 4 字节；
- 在 Go 中对字符串使用 len 将返回字节数，而不是 rune 数量。
